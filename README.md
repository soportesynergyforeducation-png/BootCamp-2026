# Bootcamp Sinergético 2026

Dashboard de escaleta y operación para el Bootcamp Sinergético 5–7 de junio 2026.

## Estructura

```
/
├── index.html      # Dashboard completo (single-file)
├── vercel.json     # Config de deploy en Vercel
└── README.md
```

## Deploy

El proyecto se despliega automáticamente en Vercel al hacer push a `main`.

- **URL producción:** asignada por Vercel al conectar el repo
- **Tipo:** sitio estático (no requiere build)
